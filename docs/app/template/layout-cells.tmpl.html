Child items, or cells, of a layout element can be easily arranged using additional alignment attributes. Note that these examples do not have the flex attribute, which allows these demos to better illustrate how cells arrange themselves within a layout container. In each demo, the center black line represents the "main axis".

Main Axis: Center

The main-axis="center" attribute will center the cells across the main axis. Note that each demo's center black line represents the "main axis". Additionally, by default each item will "stretch" the cross axis, which is explained later.

Cross Axis: Stretch (default)

Be default child cells will "stretch" the same distance as the cross axis. Note that each demo's center black line represents the "main axis", which makes the perpendicular area the cross axis.

(To stretch the cells across the main axis is where the flex attribute comes in.)
